用法：
打开游戏，进入副本界面
使用管理员方式打开命令行提示符
进入到你安装软件的根目录
输入: ./main.exe 执行

set times >>:  输入副本次数
select mod {'0': 'actives', '1': 'soul_snake', '2': 'awaking'} ('q' to exit)>>: 选择模式
0，当前的活动
1，御魂
2，觉醒


注意
1, 如果需要自定义配置，请先运行一次该程序，定位好窗口后再截图，默认都是配置好的

2，请不要在执行中移动窗口，并保持窗口激活，否则报错需要重启程序

3，活动页是数据是动态变化的，每有新活动都需要重新更新触发器

4，按q以退出, 按r以重新执行当前副本，卡死时使用

5，显示器不兼容的情况下，运行一遍程序，自己添加配置

6，不适合长线副本 （>100s）

7，未正常执行时有3分钟自重启，但仍然推荐手动重启
